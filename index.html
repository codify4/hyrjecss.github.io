Çfare eshte CSS?

CSS (Cascading Style Sheets) eshte gjuha qe perdoret per te stilizuar dhe dizajnuar faqet e internetit. Ajo ndan strukturen (HTML) nga paraqitja (CSS), duke e bere faqen me te organizuar dhe me terheqese.

Perfitimet perfshijne konsistence ne dizajn, mirembajtje me te lehte dhe pervoje me te mire per perdoruesin.

Sintaksa e CSS

Formati baze i nje rregulli CSS eshte:

selektori { property: value; }

Shembull:

p { color: blue; font-size: 16px; }

Llojet e CSS

Inline CSS: Vendos stil direkt ne elementin HTML.

<p style="color: red;">Tekst i kuq</p>

Internal CSS: Perdor tag-un <style> brenda dokumentit HTML.

<style> h1 { text-align: center; } </style>

External CSS: Ruhet ne nje file te veçante dhe lidhet permes <link>.

<link rel="stylesheet" href="styles.css">

Box Model

Box Model eshte struktura qe percakton si shfaqen elementet HTML. Ai perbehet nga:

  • Content: Permbajtja reale.
  • Padding: Hapesira midis content dhe border.
  • Border: Kufiri rreth elementit.
  • Margin: Hapesira jashte border qe ndan elementet.

Animacionet

Animacionet krijojne efekte vizuale per te permiresuar interaktivitetin.

  • .animated-square { width: 100px; height: 100px; background-color: blue; animation: moveAndChange 2s infinite; }
  • @keyframes moveAndChange { 0% { transform: translate(0, 0); background-color: blue; } 50% { transform: translate(100px, 0); background-color: red; } 100% { transform: translate(0, 0); background-color: blue; } }
