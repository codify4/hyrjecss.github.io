Hyrje në CSS

Një udhëzues për bazat e CSS për ndërtimin e faqeve të bukura web.

Çfarë është CSS?

CSS (Cascading Style Sheets) është gjuha që përdoret për të stilizuar dhe dizajnuar faqet e internetit. Ajo ndan strukturën (HTML) nga paraqitja (CSS), duke e bërë faqen më të organizuar dhe më tërheqëse.

Përfitimet përfshijnë konsistencë në dizajn, mirëmbajtje më të lehtë dhe përvojë më të mirë për përdoruesin.

Sintaksa e CSS

Formati bazë i një rregulli CSS është:

selektori { property: value; }

Shembull:

p { color: blue; font-size: 16px; }

Llojet e CSS

Inline CSS: Vendos stil direkt në elementin HTML.

<p style="color: red;">Tekst i kuq</p>

Internal CSS: Përdor tag-un <style> brenda dokumentit HTML.

<style> h1 { text-align: center; } </style>

External CSS: Ruhet në një file të veçantë dhe lidhet përmes <link>.

<link rel="stylesheet" href="styles.css">

Box Model

Box Model është struktura që përcakton si shfaqen elementët HTML. Ai përbëhet nga:

  • Content: Përmbajtja reale.
  • Padding: Hapësira midis përmbajtjes dhe kufirit.
  • Border: Kufiri rreth elementit.
  • Margin: Hapësira jashtë kufirit që ndan elementët.

Ngjyrat në CSS

Mënyrat për të zgjedhur ngjyrat përfshijnë:

  • Named Colors: P.sh., red, blue.
  • Hex Codes: P.sh., #FF5733.
  • RGB: P.sh., rgb(255, 87, 51).

Animacionet

Animacionet krijojnë efekte vizuale për të përmirësuar interaktivitetin.

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } div { animation: fadeIn 2s ease-in-out; }

Provo animacionin:

<div style="width: 100px; height: 100px; background: #007BFF; animation: fadeIn 2s ease-in-out;"></div>
